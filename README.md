# 🎲 Консольная игра «Угадай число» (Number Guessing Game)

Простая консольная игра на C# — программа загадывает случайное число от 1 до 100, а вы пытаетесь его угадать с подсказками «больше» или «меньше».

## 🚀 Возможности

- Генерация случайного числа от 1 до 100
- Подсчёт количества попыток
- Подсказки: «больше» или «меньше» загаданного числа
- Сохранение результата последней игры в текстовый файл (`Data.txt`)
- Поздравление с победой с указанием числа и количества попыток

## 🛠️ Технологии

- C# (.NET)
- Работа с консолью (`System.Console`)
- Генерация случайных чисел (`System.Random`)
- Запись в файл (`System.IO.File`)

## ⚙️ Настройка перед запуском

**Важно!** Откройте файл `Main.cs` и найдите строку:

```csharp
string path = @"YOUR_ADDRESS_FROM_FILE";
```
Замените YOUR_ADDRESS_FROM_FILE на свой путь к файлу Data.txt
