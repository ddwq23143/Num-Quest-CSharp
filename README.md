# Num-Quest-CSharp
Консольная игра на C#, в которой компьютер загадывает случайное число от 1 до 100, а игрок пытается его отгадать. После каждой попытки программа подсказывает: «больше» или «меньше». Когда число угадано, игра поздравляет победителя, показывает количество попыток и сохраняет результат в текстовый файл Data.txt.
